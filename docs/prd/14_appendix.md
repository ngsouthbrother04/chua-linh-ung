## 1. Glossary

| Term | Definition |
|---|---|
| POI | Point of Interest, a food location shown on map and QR flows |
| Single Voice Rule | Only one narration may play at any time |
| Offline-first | Core exploration works from local storage after initial sync |
| Sync Manifest | Server metadata used to decide whether client needs full sync |
| SQLite Mirror | Local mobile database tables for cached content |
| Server-side TTS | Audio files generated by backend and delivered as MP3 URLs |
